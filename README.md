# Data

This is the place where all the results of fp-stats go!

## runtime-version-information format

It's a simple JSON which ressembles this
```
{
    "Freedesktop": {
        "org.freedesktop.Platform//20.08": [
            "tld.domain.app"
        ]
    }
}
```

## runtime-arch-information format

It's a simple JSON which ressembles this
```
{
    "x86_64": [
        "tld.domain.app"
    ]
}
```
